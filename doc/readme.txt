Javadoc may be generated as follows (please do not commit the output to the repository):

1. open the eHealthConnector project in Eclipse
2. Select the eHealthConnector project in the project explorer
3. Use the Eclpise menu "Project -> Generate Javadoc ..."
4. Make sure that your Javadoc command points to javadoc.exe of your JDK bin
5. Make sure that the destination folder points to this directory (\trunk\ehealthconnector\doc)
6. Press finish
7. Open index.html

Publication location of the Online Javadoc: http://www.medshare.net/fileadmin/eHealthConnectorAPI/doc/
